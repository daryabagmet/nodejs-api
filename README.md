## Camp activity board

### API Reference

#### Get all camps

```http
  GET /api/v1/camps
```

#### Get camp

```http
  GET /api/v1/camps/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**.                     |

#### Create new camp

```http
  POST /api/v1/camps
```

#### Delete camp

```http
  DELETE /api/v1/camps/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**.                     |
